# Ideation Validator

This project provides a simple ideation validator that loads ideas from a database and provides validation metrics for each idea.

## Usage

1. Load ideas from a database using the `load_ideas` function.
2. Create an instance of the `IdeationValidator` class with the loaded ideas.
3. Use the `get_validation_metrics` method to retrieve validation metrics for a specific idea.
4. Use the `refresh_data` method to simulate data refresh.
5. Use the `get_credits` method to retrieve credits for a specific idea.
